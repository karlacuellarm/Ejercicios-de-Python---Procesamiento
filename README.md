# Ejercicios-de-Python---Procesamiento
En este Taller de Python logramos aprender mucho a cerca de diferentes temas que son las bases de cualquier codigo y mejoramos nuestras habilidades y entendimiento de ellas.

1. Cadenas: experimentamos con Cadenas para entender mejor que son, como y cuando se usan y que tanto podemos hacer con ellas, empezamos desde los mas simple para ir avanzando.
2. Tuplas: Las tuplas nos ayudan a entender en el concepto de datos como se pueden ver y almacenar diferentes tipos de datos correspondientes a algo y como podemos usarlos en nuestro beneficio.
3. Listas: en este entendemos que son las listas y como podemos usarlas con fines especificos utilizando sus mismos elementos como los indices.
4. Conjuntos: con este adem√°s de aprender sobre conjuntos, entendemos algunas diferencias basicas con los anteriores como con los objetos repetidos.
5. Diccionarios: de nuevo como con los anteriores aprendemos de ellos y sus operaciones pero sobre lo que lo distingue y por lo mismo como se usa diferente por ejemplo con el manejo de llaves.
6. Condiciones: En este notebook entendemos mas sobre algunas de las operaciones que estan en python y como podemos usar estas en conjunto con los condicionales para mejorar nuestro codigo.
7. Bucles: Los bucles son importantes pues nos ayudan en gran medida a optimizar nuestro programa en cuanto a la repeticion de instrucciones u operaciones.
8. Funciones: aprendemos sobre la definiciones y funciones y las partes importantes de como estas se pueden declarar y usar.
9. Clases: las clases nos ayudan al orden y es muy importante aprender como se crean y usan para facilitar nuestro codigo.
